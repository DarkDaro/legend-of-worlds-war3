# Механики Legend of Worlds

## Удача
Все источники складываются в одно значение. Удача сильнее помогает низким базовым шансам, слабее — высоким. Никогда не толкает шанс выше 100%.

```
Итоговый шанс = Базовый% + Удача × (1 − Базовый% / 100)
```

Примеры (удача = 40):
- База 5% → 5 + 40 × 0.95 = 43%
- База 15% → 15 + 40 × 0.85 = 49%
- База 50% → 50 + 40 × 0.50 = 70%
- База 90% → 90 + 40 × 0.10 = 94%

## Магический крит
Отдельный множитель для заклинаний. Работает независимо от физического крита.
```
Маг. урон = Базовый урон × Маг. крит (1.5 / 2.0 / 2.5)
```

## Магический вампиризм
Восстановление HP от магического урона способностей.
```
Хил = Маг. урон × Вампиризм%
```

## Физический вампиризм
Несколько источников складываются мультипликативно.
```
Итого = 1 − (1−a) × (1−b) × (1−c)...
```

## Блок урона
Шанс полностью заблокировать физическую атаку (0 урона). Срабатывает до брони. Не складывается — берётся наивысшее значение.

## Сопротивление оглушению
Снижает длительность CC-эффектов. Не снижает урон.
```
Длительность = Базовая × (1 − Сопрот.% / 100)
```

## Снижение лечения
Дебафф 'A13M' снижает лечение на 50%. Несколько источников не складываются — берётся наибольшее.
```
Лечение = Базовое × (1 − Снижение%)
```

## Скипетр Владыки
Уникальный предмет, усиливающий ультимейт каждого героя. Эффект индивидуален: от доп. урона до шанса мгновенной перезарядки. См. карточку героя.
