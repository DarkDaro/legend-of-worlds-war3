# Legend of Worlds — энциклопедия

Сайт-энциклопедия кастомной карты **Legend of Worlds** для Warcraft III.

🔗 [darkdaro.github.io/legend-of-worlds-war3](https://darkdaro.github.io/legend-of-worlds-war3)

## Что на сайте

### Герои (25 карточек)
- Полные характеристики: HP, мана, атака, защита, скорость, реген, дальность боя
- Таблица атрибутов: Сила / Разум / Ловкость — базовые значения и прирост за уровень
- Рекомендуемые сборки предметов (6 слотов) — динамический рендер из базы данных
- Навигация prev/next между героями
- 6 героев с полными данными (Паладин, Адмирал, Друид, Киборг, Громовержец, Иллюзионист)

### Предметы (~200 позиций)
- Каталог по категориям магазина с поиском и фильтрацией
- Рекурсивные деревья рецептов — каждый предмет раскрывается до базовых компонентов
- Перекрёстные ссылки «какие герои собирают этот предмет»
- Теги типов: оружие, броня, аксессуар, босс-дроп, расходуемый и т.д.
- Избранное — добавление предметов в личный список
- Детальная панель: статы, рецепт, компоненты, ссылки на сборки героев

### Калькулятор сборок
- 6 слотов, автоподсчёт стоимости с разбивкой на базовые покупки и рецепты
- Пометка продающихся предметов (возврат золота)
- Дерево компонентов — раскрывается кликом
- Сохранение сборки в localStorage
- Автозагрузка рекомендованной сборки при выборе героя

### Конструктор сборок
- Многостадийный конструктор — несколько этапов покупки предметов
- Генерация JASS-кода для внедрения в карту
- Экспорт кода для Warcraft III World Editor

### AI-сборки
- 19 групп, 408 стадий ботов — сгенерированные из скриптов карты
- Просмотр по группам и героям

### Монстры
- 8 боссов — статы, способности, стадии HP, лорные описания
- 25 крипов — краткие карточки

### Механики и режимы
- Формулы: удача, крит, вампиризм, броня, магзащита, мотивация, уклонение
- Кнопки «Рассчитать» — переход к калькуляторам прямо из карточек механик
- 9 уникальных режимов игры

### Калькуляторы
- Броня — расчёт снижения урона по формуле карты
- Формулы — 4 вкладки: урон, крит, вампиризм, мотивация
- Сравнение героев — выбор двух героев, таблица характеристик рядом

### Лор
- История мира — 5 эпох, от Древних до Восстания
- Таймлайн с ключевыми событиями
- Призыв к действию

## Особенности

- **Рекурсивные деревья рецептов** — каждый предмет раскрывается до базовых компонентов, неограниченная глубина
- **Перекрёстные ссылки** — от предмета к героям, от героя к предметам, от механики к калькулятору
- **Динамический рендер** — статы, сборки и навигация героев генерируются из JS-базы данных, не дублируются в HTML
- **Неон + стекломорфизм** — полупрозрачные карточки с размытием, градиентные рамки, свечением при наведении, анимации при скролле
- **Мобильная адаптация** — все страницы работают на телефоне, адаптивная сетка, гамбургер-меню
- **Глобальный поиск** — Ctrl+K ищет по героям и предметам
- **Единый источник констант** — все формулы карты в `game-constants.js`, калькуляторы импортируют из него
