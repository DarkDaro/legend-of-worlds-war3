Игровые режимы

Перед началом игры хост выбирает режимы, которые меняют правила матча.

Урон заклинаний = 50%

Уменьшает урон всех заклинаний вдвое. Влияет только на магический тип урона — автоатаки и чистый урон не затронуты. Снижает роль магов и смещает баланс в сторону автоатакеров.

Боссы

Включает боссов на карте. Без этого режима предметы, выпадающие с боссов, недоступны — а значит, стадии сборки, требующие босс-предметы, будут пропущены.

Потеря золота

При смерти от вражеского героя вы теряете часть золота. Добавляет риск и ценность выживания. Без режима — смерть не стоит ресурсов.

Турбо

Ускоренный режим: повышенный опыт, больше золота, быстрее темп. Для быстрых матчей.

Без книг

Отключает книги и руны прокачки характеристик. Убирает «пей-ту-вин» — все решает только сборка и скилл.

Командные награды

Дополнительные награды за командные действия: убийства, ассисты, контроль. Поощряет командную игру вместо соло-кари.

Соло режим

Отключает командный режим. Каждый играет сам за себя. Изменяет механики для соло-формата.

Бан героев

Два игрока из двух команд по очереди банят героев. Забаненные герои недоступны для выбора в матче.

Случайный герой

Каждый игрок получает случайного героя вместо выбора. Уравнивает шансы и добавляет элемент неожиданности.
