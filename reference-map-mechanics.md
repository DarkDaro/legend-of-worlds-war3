## Механики урона

### Удача (Luck)
- Влияет на шанс срабатывания случайных эффектов: крит, оглушение, уклонение, предметы, пассивки
- Формула: `Итоговый шанс = Базовый% + Удача` (простое сложение, кап 100%)
- Используется через `ChanseLucky(unit, baseChance)` — добавляет GetUnitLucky к базовому шансу
- Источники: предметы (+5, +10, +15)

### Магический крит
- Отдельный множитель крита для заклинаний, работает независимо от физического
- Множители: 1.5x / 2.0x / 2.5x

### Усиление маг. урона (SpellDMG)
- Увеличивает урон заклинаний (категории 1 и 2)
- Формула: `Маг. урон = Базовый × (1 + SpellDMG)`
- Аддитивное сложение всех источников
- Примеры: предмет даёт +0.10 (10%), скипетр +1.00 (100%)

### Чистый урон (CLEAN / MODE_UNMODIFIED)
- Тип урона, который **не** обрабатывается модификаторами
- Не усиливается SpellDMG, не блокируется маг. защитой, не учитывает вампиризм, не вызывает пассивок
- Используется для: возврата урона, порчи, гарантированных эффектов, убийственных способностей

### Типы урона (DamageCategory)
| Категория | Описание |
|---|---|
| PHYSICAL (1) | Физический |
| MAGICAL (2) | Магический — усиливается SpellDMG |
| CLEAN (4) | Чистый — без модификаторов |

### Режимы урона (DamageMode)
| Режим | Описание |
|---|---|
| MODE_ATTACK (1) | Обычная атака |
| MODE_MELEE_ATTACK_EMULATION (2) | Эмуляция мили атаки |
| MODE_RANGE_ATTACK_EMULATION (3) | Эмуляция ренж атаки |
| MODE_SPELL (4) | Способность/предмет |
| MODE_UNKNOWN (5) | Неизвестный |
| MODE_UNMODIFIED (6) | Без обработки модификаторами |

## Механики защиты

### Магическая защита (ItemRezCal)
- Снижает входящий магический урон
- **Мультипликативная** формула с убывающей полезностью
- `Защита = 1 − (1−a) × (1−b) × (1−c)...`
- Пример: два предмета по 20% = 36% защиты, не 40%
- Предметы с ResId: Огненный щит (15%), Рыцарский щит (10%), Шлем дракона (5%), Доспехи дракона (25%), Доспехи полководца (20%), Антимагические доспехи (30%)

### Блок урона (MagicBlock)
- Шанс заблокировать часть урона (любого типа, включая атаки и заклинания)
- `сумма = базовый_блок + блок_от_предметов + бонус_усиления`
- `NegateDamage(unit, урон × сумма)` — возвращает часть урона
- Визуальный эффект: AbsorbManaBirthMissile

### Сопротивление оглушению (ItemStunReduceCal)
- Снижает длительность оглушения, замедления, контроля
- **Мультипликативная** формула: `1 − (1−a)×(1−b)×(1−c)...`
- Применяется в BashUnit: `last = last × (1 − ItemStunReduceCal(target))`
- Предметы: Рыцарский щит (30%), Шлем штормового рыцаря (25%), Щит штормового рыцаря (45%)

### Сопротивление сожжению маны (ItemManaBurnReduceCal)
- Снижает количество маны, выжигаемой способностями
- **Мультипликативная** формула (как маг. защита)
- `Сожжено = Базовое × (1 − Защита)`
- Предметы: Рыцарский щит (10%), Антимагические доспехи (15%)

### Снижение лечения (дебафф A13M)
- Уменьшает входящее исцеление на 50%
- Несколько источников **не** складываются — берётся наибольшее
- Проверяется в SpellHealUnit и HealUnit

## Механики лечения

### Усиление маг. лечения (HealBonus)
- Увеличивает исцеление от способностей
- Формула: `Лечение = Базовое × (1 + HealBonus)`
- Применяется **до** проверки на снижение лечения
- Аддитивное сложение

### SpellHealUnit vs HealUnit
- **HealUnit** — основная функция лечения, учитывает HealBonus + дебафф A13M
- **SpellHealUnit** — магическое лечение, учитывает только дебафф A13M (без HealBonus)

## Вампиризм

### Физический вампиризм (ItemVampCal)
- Восстановление HP от физического урона автоатаками
- **Мультипликативная** формула: `1 − (1−a)×(1−b)×(1−c)...`
- Усиление вампиризма (LifeSteal) добавляется к базовому: `итого = базовый + предметный + бонус`

### Магический вампиризм (MagicVamp)
- Восстановление HP от магического урона заклинаниями
- Отдельная система от физического
- Хил через SpellHealUnit

## Порча (Corruption / WCorraption)

Стакающийся дебафф, применяемый способностью «Порча» (некромант):

1. **Бонусный урон**: каждый стак +20% к получаемому урону (чистым типом, MODE_UNMODIFIED)
   - `bonusDamage = DS_Damage × (0.20 × стаки)`
   - Срабатывает при **любом** полученном уроне (атаки, заклинания)
2. **DPS**: периодический урон = `1.0 × Интеллект(кастер) × стаки`
3. **Длительность**: обновляется при каждом новом стаке
4. **Снятие**: при истечении таймера, смерти цели или иммунитете к магии

## Возврат урона (JaggerReturnPass)

Предметы возвращают часть полученного урона атакующему:
- Доспех горного короля (I01L): 20%
- Щит джаггернаута (I02D): 25%
- Нерубский доспех (I0A5): 30%
- Улучшенный щит джаггернаута (I0B6): 60%

Возврат = `полученный урон × процент`, тип CLEAN + MODE_UNMODIFIED (не блокируется, не усиливается)

## Система баффов (BuffSystem)

Кастомная система через ауру Торнадо:
- Каждый бафф имеет свойства: длительность, позитивный/негативный, снимается при смерти, можно развеять
- `ApplyBuffToUnit(unit, abilityBuffId, buffId, duration, stackDuration, isPositive, removeUponDeath, canBeDispelled)`
- `DispellBuff(unit, abilityBuffId)` — снять конкретный бафф (если canBeDispelled)
- `DispellAllBuffs(unit, positive, negative)` — снять все баффы выбранного типа
- Можно добавить абилку к баффу через `BuffAddAbility()` — удалится вместе с баффом
- Стакание длительности: stackDuration=true → продлевает, false → обновляет

## Сборка предметов в 1 клик (ItemBuild)

Стадийная система автоматической сборки:
- Каждый герой привязан к группе сборки (`AddUnitTypeToBuild`)
- Группа содержит стадии (до 6 предметов в каждой)
- Стадия требует: уровень героя, золото, отсутствие требований боссов (если боссы выключены)
- Процесс: продажа старых → списание золота → выдача новых предметов
- Бот-система: автоматически переходит между стадиями при наличии золота и уровня

## Баш (BashUnit)

Оглушение при атаке:
- Шанс срабатывания: `ChanseLucky(caster, baseChance)` (учитывает удачу)
- Проверка CD: `IsBashCooldown(caster, itemId)` — предотвращает двойные срабатывания
- Длительность: снижается сопротивлением оглушению `last × (1 − ItemStunReduceCal)`
- Мили: физический урон, ренж: магический урон

## Формулы сложения — сводка

| Тип | Формула | Пример |
|---|---|---|
| Аддитивное | a + b + c | SpellDMG, HealBonus, Lucky |
| Мультипликативное | 1−(1−a)(1−b)(1−c) | Маг. защита, Вампиризм, Сопрот. оглушению, Сопрот. сожжению маны |
| Наибольшее | max(a, b, c) | Снижение лечения (A13M) |
